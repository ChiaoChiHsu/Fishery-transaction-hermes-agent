# Fishery-transaction-hermes-agent
It's an hermes agent system designed for fishery transaction data analysis. The system is able to fetch data on Public OpenData exchange platform and plot charts for visualizations.
## Quick start
Install hermes webui
```bash
git clone https://github.com/nesquena/hermes-webui.git hermes-webui
cd hermes-webui
```
